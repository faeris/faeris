*)查询一个状态当前是处于打开还是关闭状态
    GLboolean glIsEnalbled(Glenum capability)
    参数：capability值如下
      1) GL_LINE_STIPPLE (直线点画功能)

      5) GLRESCALE_NORMAL (对法线向量进行缩放)
